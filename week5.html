Week 5

In deze week kregen we theorie over Visual Studio Code (VS Code). Aangezien ik hier persoonlijk nog nooit mee gewerkt had, was het in het begin best uitdagend. Gelukkig mochten we ChatGPT gebruiken om te helpen bij het omzetten van onze ontwerpen naar code.

Onze taak was om de ontwerpen die we in Figma hadden gemaakt samen met ChatGPT te vertalen naar HTML-code. Door deze samenwerking konden we stapsgewijs aanpassingen doen totdat het resultaat steeds meer leek op ons oorspronkelijke ontwerp.

Daarnaast bood deze fase de mogelijkheid om zelf extra functionaliteiten toe te voegen, zoals bijvoorbeeld een live klok of andere interactieve elementen. Dit maakte het project niet alleen visueel representatief, maar ook functioneel en dynamisch.

Deze week stond vooral in het teken van digitalisering, codering en interactie, waarbij we leerden hoe je ontwerpen uit een grafisch programma kan vertalen naar werkende webpagina’s. Het werken met VS Code en ChatGPT gaf veel inzicht in het proces van prototyping naar werkende code en het toepassen van creativiteit in de webontwikkeling.
